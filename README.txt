PHE Calculator
A web-based calculator that calculates the PHE (Phenylalanine) values of a given food input and stores it in a MySQL database.


Introduction
PHE Calculator is a Flask-based web application that allows users to calculate the PHE values of a given food input. The application uses a MySQL database to store the calculated values and provides a user-friendly interface to view and manage the data.

Features
Calculates PHE values for a given food input
Stores calculated values in a MySQL database
Allows users to store food PHE values in 100g in a separate table
Enables users to calculate daily PHE intake based on stored values
Displays daily intake history for every day
Built using Flask, Bootstrap, and CSS for a responsive and user-friendly interface
Installation
To install the project, follow these steps:

Clone the repository using git clone https://github.com/PavicicJadranko/PHE.git
Install the required dependencies using pip install -r requirements.txt
Create a MySQL database and update the config.py file with the database credentials
Run the application using flask run
Usage
To use the application, follow these steps:

Open the application in a web browser
Enter the food input and click the "Calculate" button to calculate the PHE values
View the calculated values and store them in the database if desired
Use the "Daily Intake" feature to calculate the daily PHE intake based on stored values
View the daily intake history for every day
